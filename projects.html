Post-Training Quantization Analysis

Academic Project

Implemented and analyzed post-training quantization techniques on MobileNetV2 for image classification (Rock-Paper-Scissors dataset) using TensorFlow. Compared model efficiency and accuracy before and after quantization, including integer quantization, n-bit clustering, and linear quantization.

  • Original Model (Float32): 2,227,715 params, 8.50MB, 80.11% accuracy, 0.1369s inference.
  • Post-Training Integer Quantization (int8): 4.40MB, 71.50% accuracy, 0.0093s inference (14× speedup), 8.5MB → 4.4MB, 7× faster, accuracy drop ~9%.
  • n-bit Clustering Quantization: 2-bit (7.77×), 4-bit (7.73×), 8-bit (6.96×) compression; quantization error (MSE): 2-bit: 76.29, 4-bit: 59.21, 8-bit: 46.66.
  • Linear Quantization (int8/int16): int8: 7.78× compression, 73.12% acc; int16: 3.91× compression, 79.84% acc; minimal performance loss for int16.
